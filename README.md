# Automatic-Noise-Level-Monitor-Controller-System

As I continue to dive deeper into the world of embedded systems, I wanted to share an academic project that truly solidified my passion for hardware-software integration: the Automatic Noise Level Monitor & Controller System.
Noise pollution in educational and professional environments can severely impact focus. To tackle this, I designed a microcontroller-based solution that actively monitors ambient sound and alerts users when levels exceed acceptable limits.

🔧 Technical Stack & Tools:
Hardware: Arduino (ATmega328), Analog Sound Sensors, 20x4 LCD, Pushbuttons, Alert Peripherals (Buzzer/LEDs)
Software: Embedded C / C++
Design: Circuit schematic design and PCB layout implementation

💡 Key Features & Contributions:
Dynamic Thresholding: Programmed the firmware to allow users to select customized decibel limits based on the environment directly from the hardware UI (e.g., 30dB for a Library, 60dB for a School, 75dB for a College).
Multi-Sensor Data Acquisition: Interfaced multiple analog sound sensors to ensure accurate, multi-directional audio sampling in real-time.
End-to-End Development: Took the project from breadboard circuit conceptualization to coding the underlying logic, and finally, assembling the physical PCB prototype.

This project was a fantastic hands-on experience in sensor calibration, UI programming on limited displays, and troubleshooting physical hardware components to create a reliable, working prototype.
